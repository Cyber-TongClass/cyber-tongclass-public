# 清华通班学生组织展示设计

## 目标

在“关于通班”页面的“学生组织”页签中，按北大通班现有的卡片版式补充清华通班的学生组织信息，并提供一个紧凑的可展开科研信息区。

## 页面结构

清华通班区块位于北大通班区块之后，保留现有标题层级与间距。主体采用与北大相同的响应式 `1fr / 2fr` 网格：左侧是带酒红色左边框的“通班自治委员会”主卡；右侧是两列网格，展示“通班文化建设委员会”“通班信息网络技术委员会”和“通班顾问委员会”。每张卡包含用户提供的简介和职责列表。

组织网格下方放置一个紧凑的灰色文字折叠按钮，标签为“通班学术发展与科研实践”。按钮含小号 `ChevronDown` 图标，在展开时旋转；按钮通过 `aria-expanded` 和关联内容区域标识其状态。展开区域显示三张部门卡样式的卡片：“通班学术发展与科研实践”“通班科研成果”和“通班小班主任制度”，并完整呈现用户提供的介绍与要点。

## 实现边界

仅修改 `src/app/about/page.tsx`。清华组织及科研内容均定义在该组件内的静态数据中，通过映射生成卡片。组件保留客户端边界以维护折叠状态，不调用 Convex，也不改动现有北大内容或后端代码。

## 响应式与可访问性

桌面端沿用现有卡片尺寸、阴影、酒红强调色和悬停效果；小屏幕按现有 Tailwind 断点退化为单列。折叠控件使用原生 `button`，支持鼠标与键盘触发，图标的旋转仅表达视觉状态且不替代文本状态。

## 验证

新增面向页面源文件的静态检查，断言四个组织标题、折叠控件、三个折叠内容标题、`aria-expanded` 和 `ChevronDown` 均存在；先运行检查观察其在实现前失败，再实施最小变更使其通过。随后运行 `npm run lint`。
